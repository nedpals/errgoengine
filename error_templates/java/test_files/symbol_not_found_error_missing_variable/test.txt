# SymbolNotFoundError
The program cannot find variable "a"
```
    public static void main(String[] args) {
        System.out.println(a);
                           ^
    }
}
```
## Steps to fix
### Create a variable.
Create a variable named "a". For example:
```diff
public class Program {
    public static void main(String[] args) {
-         System.out.println(a);
+         String a = "";
+         System.out.println(a);
    }
}
```
