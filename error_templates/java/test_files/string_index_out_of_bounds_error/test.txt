# StringIndexOutOfBoundsException
This error occurs when attempting to access an index beyond the length of a string.
```
        // Attempting to access an index beyond the string's length
        char character = text.charAt(20);
                                     ^^
        System.out.println(character);
    }
```
## Steps to fix
### 1. Ensure the index is within the string length
1. Check that the index used for accessing the character is within the valid range of the string length.
```diff
+ if (index < text.length()) {
    char character = text.charAt(20);
    System.out.println(character);
+ } else {
+    System.out.println("Index out of range.");
+ }
```
