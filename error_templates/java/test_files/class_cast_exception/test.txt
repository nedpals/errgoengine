# ClassCastException
This error occurs when there is an attempt to cast an object to a type that it is not compatible with.
```
        Integer number = (Integer) obj;
    }
    ^
}

```
## Steps to fix
### Use the correct type for casting
Ensure that the casting is done to the correct type. In this case, you should cast to `String` instead of `Integer`.
```diff
- Integer number = (Integer) obj;
+ String text = (String) obj; // Change the type to String
```
